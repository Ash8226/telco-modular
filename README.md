# Singtel Smart Upgrade Prototypes

Interactive HTML prototypes for exploring how Singtel could improve the experience of renewing a mobile plan, upgrading a device, and making plan value more transparent for an existing customer.

## Prototypes

- `singtel-smart-upgrade-mvp.html` - AI-assisted renewal and device upgrade journey guided by Mira.
- `singtel-modular-plan-builder.html` - Modular Plan Builder where Praveen can drag and drop plan modules, select a phone, and view a live monthly cost estimate.
- `singtel-modular-plan-builder-wireframe.html` - Wireframe version of the Modular Plan Builder concept.

## How To View Locally

Open any of the HTML files directly in a browser.

For the second prototype, open:

```text
singtel-modular-plan-builder.html
```

Keep the image and video assets in the same folder as the HTML files so the prototype can load them correctly.

## Publishing With GitHub Pages

1. Upload this folder to a GitHub repository.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the main branch and root folder.
6. Save, then wait for GitHub to generate the public URL.

The prototype will be available at a GitHub Pages URL such as:

```text
https://your-username.github.io/your-repository-name/singtel-modular-plan-builder.html
```
